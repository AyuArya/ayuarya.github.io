Luas Segitiga

Rumus Luas Segitiga yaitu:

L = 1/2 x a x t

Dimana:

L = Luas

a = Panjang Alas

t = Tinggi



Hitung Luas Segitiga

Keliling Segitiga

Rumus Keliling Segitiga yaitu:

K = S1 + S2 + a

Dimana:

K = Keliling

a = Panjang Alas

S1 = Panjang sisi 1

S2 = Panjang sisi 2


Hitung Keliling Segitiga

Luas Jajargenjang

Rumus Luas Jajargenjang yaitu:

L = a x t

Dimana:

L = Luas

a = Panjang Alas

t = Tinggi


Hitung Luas Jajargenjang

Keliling Jajargenjang

Rumus Keliling Jajargenjang yaitu:

K = 2 x (a + b)

Dimana:

K = Keliling

a = Panjang Alas

b = Panjang sisi b


Hitung Keliling Jajargenjang
